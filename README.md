
# Kanshi RPC

Shows what you're watching on Kanshi as your Discord Rich Presence status.

---

## Requirements

- **Python 3.8+** — [download here](https://www.python.org/downloads/)
- **Discord desktop app** running on your PC

---

## Setup (do this once)

### Step 1 — Install dependencies

Open a terminal in this folder and run:

```
pip install -r requirements.txt
```

### Step 2 — Generate your SSL certificate

> This is required because Kanshi is served over HTTPS, so the local helper needs a secure connection too. Each user generates their own certificate — it stays on your machine and is never shared.

**Option A (Recommended) — mkcert (no browser warning, easier):**

```
# Install mkcert
winget install FiloSottile.mkcert       # Windows
brew install mkcert                      # macOS

# Generate your cert
python gen_cert.py
```

**Option B — Self-signed (fallback, no extra install needed):**

```
python gen_cert.py
```

Then, when prompted, visit `https://localhost:8080` in your browser and click **Advanced → Proceed to localhost** once to trust it.

### Step 3 — Create a Discord Application

1. Go to [discord.com/developers/applications](https://discord.com/developers/applications)
2. Click **New Application**, give it any name (e.g. "Kanshi")
3. Copy the **Application ID** (also called Client ID)

### Step 4 — Run the helper

```
python discord_helper.py
```

Keep this terminal open while you browse Kanshi.

### Step 5 — Connect in Kanshi

1. Go to **Settings → Integrations** on [anistream-dun.vercel.app](https://anistream-dun.vercel.app)
2. Paste your **Client ID** and click **Save**
3. The helper will connect and your Discord status will update automatically!

---

## Auto-start on Windows boot (optional)

Press `Win+R`, type `shell:startup`, press Enter. Create a `.bat` file there:

```bat
@echo off
python "C:\path\to\discord-helper\discord_helper.py"
```

Replace the path with wherever you extracted this folder.

## Files in this folder

| File | Purpose |
|---|---|
| `discord_helper.py` | The main helper app — run this |
| `gen_cert.py` | One-time cert generator — run once during setup |
| `requirements.txt` | Python dependencies |
| `localhost.pem` | Your SSL certificate (generated by you, stays on your machine) |
| `localhost-key.pem` | Your private key (generated by you, **never share this lol**) |
